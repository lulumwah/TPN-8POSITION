TP N°8 Positions

1-¿Cuál es la diferencia principal entre position relative y position absolute?

La diferencia de position relative y absolute es que en la relative el elemento se desplazará relativo a su posición normal en el flujo del documento, pero el espacio ocupado por el elemento no cambiará.
Y en la absolute permite colocar un elemento en una posición específica dentro de su contenedor principal o el documento en su totalidad.

2-¿Qué valor de position se utiliza por defecto para los elementos HTML?

Los valores que se utilizan para position son top, bottom, left o right

3-¿Qué propiedad se utiliza para establecer la capa de apilamiento de elementos superpuestos?

La propiedad que se utiliza para esto es Z-Index

4-¿Cuándo se utiliza position fixed en CSS?

Position fixed se utiliza para fijar un elemento

5-¿Cómo se comporta un elemento con position sticky al desplazarse?

El sticky Es tratado como un elemento posicionado relativamente hasta que su bloque contenedor cruza un límite establecido

6-¿Cuál es el valor de position adecuado para un elemento que se desea que siga el flujo normal del documento?

El valor position static es el que sirve para esto

7-¿Cuál es la forma más común de mover un elemento con position relative?

La forma mas comun para mover un elemento con position son los valores top y left

8-¿En qué casos se puede utilizar z-index para controlar el orden de apilamiento de elementos?

El z-index lo podemos utilizar para apilar cajas una encima de la otra y poder cambiar el orden.

9-¿Qué sucede cuando dos elementos tienen el mismo valor de z-index? Prueba su resultado y responde.

Lo que sucede es que se superponen uno arriba del otro
